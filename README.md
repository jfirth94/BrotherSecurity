# BrotherSecurity & Electricals Website

This repository contains the HTML, CSS, and assets for the official **BrotherSecurity & Electricals** website.

## Overview

The site provides information about our fire and security services for both domestic and commercial clients, showcasing:

- Fire Detection Systems
- Advanced CCTV
- Access Control
- Emergency Lighting
- Intruder Systems
- 24/7 Priority Support

The site also includes a gallery of recent installations and certifications.

## Technologies Used

- HTML5
- CSS3
- Responsive design (Flexbox & CSS Grid)
